以下是在 INSA 的常用词汇，欢迎补充。


| 词汇               | 解释                                   |
|-------------------|---------------------------------------|
| Ameli             |  法国健康保险                         |
| Carte Vitale      |  法国健康保险卡                       |
| CEAM (Carte Européenne d'Assurance Maladie) | 欧洲健康保险卡，可在欧洲其他国家使用 |
| CAF               | Caisse d'allocations familiales，家庭津贴基金，可以申请住房补贴                         |
| OFII                      | 法国国家移民和难民局，负责办理签证、居留许可等手续 |
| VLS-TS (visa de long séjour valant titre de séjour) |   等同于居留许可的长期居留签证（仅限第一年）                |
| RIB (Relevé d'Identité Bancaire) | 银行账户信息，用于转账、自动扣款等 |
| Titre de Séjour | 居留许可，也称长居卡（来法一年后需申请） |
| BdE (Bureau des Elèves)  | 学生处                    |
| Carte VA (Vie Associative) |   学校生活卡（不是学生卡），可享受学校洗衣、酒吧优惠等 |
| Crous (Centre régional des œuvres universitaires et scolaires)             | 公共行政机构，提供食堂、住宿等服务 |
| K-Fêt             | INSA 的酒吧                            |
| RI               |  INSA 大食堂                            |
| PD (Part Dieu)    | 有火车站和购物中心                      |
| 小家              | INSA 南边 1km 左右的家乐福            |
| 中家              | Villeurbanne 中心的家乐福              |
| 大家              | Part Dieu 的家乐福                     |
| 超家              | IKEA 大路对面的家乐福                  |
