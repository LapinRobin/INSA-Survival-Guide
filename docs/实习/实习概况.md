# 实习

INSA 的实习主要分工人实习和专业实习。


- 工人实习在大一或大二的暑假，时长一个月。

- 专业实习贯穿大三至大五，安排因专业而异。

来源：[学校官网信息](https://www.insa-lyon.fr/fr/stages-et-alternance){:target="_blank"}

## 为什么要实习

这个部分包含个人观点，仅供参考。

- 因为学校规定，不实习就毕不了业。
- 体会打工人的苦与乐。
- 知道自己不喜欢做什么。
- 提前步入社会，了解公司的运作方式。
- 积累工作经验，增加自己的竞争力。

## 如何找实习

- JobTeaser 上有很多实习信息，可以根据自己的意向筛选。
- 有些公司会在校园里举办宣讲或招聘会，可以去听听。
- 找学长学姐打听，看看他们在哪里实习过。
- 有些学长学姐会在群里发实习信息，可以关注一下。
