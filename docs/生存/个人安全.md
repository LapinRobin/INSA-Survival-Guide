# 个人安全
里昂市2022年被评为法国危险的城市，2023年退居第五位。[链接](https://ville-data.com/delinquance/classement-des-villes-les-plus-dangereuses-de-france)

INSA 里昂位于 La Doua 大学城。学校里有校警巡逻，比周边地区安全，但是晚上仍不建议单独外出游荡。

## 人身安全
独自在家时，尽量大门反锁。

外出避免治安较差的街区，如 Guillotière。

如遇抢劫，不要反抗，保持冷静，远离危险后再报警。
## 财产安全

即使出门时间很短（例如下楼扔垃圾），也要记得锁门。

外出少带大量现金，乘坐公共交通时，一定要管好手机和钱包。
