# 前言

## 这是什么

这是一份指南，旨在让你更好地度过在 INSA[^1] 的留学时光。

## 为什么需要这份指南

留学的日子里，我们在生活、学习中会遇到各种各样的问题。自己解决有困难时，我们会向同届同学或是学姐学长求助。

但这种方式主要有两个问题：

1. 信息不准确。口头或短字回答常常模糊，过时或错误。

2. 沟通成本高。提问者需要等待回应。重复问题增加答者负担。

这份指南则致力于提供准确无误的知识，做到知识共享。


## 如何使用

你可以在左侧菜单栏里选择你想了解的条目，也可以直接搜索关键词。
