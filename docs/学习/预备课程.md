# 预备课程

## 班级分类

一个年级的学生分为若干种班级。中国同学一般选择 ASINSA[^1] 或 SCAN[^2]。
国际班之外还有其他班级供选择：Classique, INS'AVENIR, Sport de Haut Niveau (SHN)[^3]


### ASINSA

俗称亚洲班，一半法国同学，一半亚洲同学。亚洲同学主要来自中国和越南。


### SCAN

俗称英语班，因为授课语言是英语。近些年班里法国籍学生增多，国际氛围有所减弱。2021届 SCAN 班里只有3名国际生。
